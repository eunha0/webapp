# 과제 상세 모달 - 하단 버튼 추가 완료

## 📋 변경 사항 요약

### 문제점
교사의 "나의 페이지"에서 과제 카드를 클릭하면 열리는 상세 정보 모달에서:
- **X 버튼**과 **출력 버튼**이 모달 상단에만 위치
- 과제 내용이 길어 스크롤이 필요한 경우, 버튼이 화면 밖으로 사라짐
- 닫기 또는 출력을 위해 다시 스크롤을 올려야 하는 불편함 발생

### 해결 방법
✅ **모달 하단에 Sticky 버튼 바 추가**
- 스크롤 위치와 관계없이 항상 하단에 버튼이 표시됨
- "닫기" 버튼과 "출력" 버튼을 하단에 배치
- 원래 상단의 "출력" 버튼도 유지 (사용자 편의성)

### 주요 CSS 클래스
- `sticky bottom-0`: 스크롤 시에도 항상 하단에 고정
- `bg-white`: 흰색 배경 (콘텐츠와 구분)
- `border-t-2 border-gray-200`: 상단 경계선
- `flex justify-end gap-3`: 오른쪽 정렬, 버튼 사이 간격
- `px-6 py-3`: 넉넉한 패딩 (클릭 영역 확보)

### 버튼 스타일
1. **닫기 버튼**
   - 회색 배경 (`bg-gray-200`)
   - 회색 텍스트 (`text-gray-700`)
   - Hover 효과: 진한 회색 (`hover:bg-gray-300`)

2. **출력 버튼**
   - 녹색 배경 (`bg-green-600`)
   - 흰색 텍스트 (`text-white`)
   - Hover 효과: 진한 녹색 (`hover:bg-green-700`)
